Adding users and registering:

The application allows you to register new users with the USER role and then log in with their help, To register, select the "Register" button in the navigation bar and complete the form, the connection to the server is not encrypted therefore the password has no password requirements, it can be the simplest (the password is saved in the database after hashing), after correct registration, a confirmation email will be sent to the address provided in the form

By default, the application has several users with which you can log in

  • Admin:
    mail: admin@gmail.com
    password: admin
  • Teacher:
    mail: teacher@gmail.com
    password: teacher
  • Random users are also created whose login and password are email.

After logging in, you can also add individual users using the form available in the "Add student" navigation bar.

Giving ratings:

After logging in as a user with the ADMIN, TEACHER or PRINCIPAL role, we get the opportunity to assign grades to students, to do this, select "students list" while logged in, search for the student and click "+" to expand the details, enter the student's profile and add grades there
